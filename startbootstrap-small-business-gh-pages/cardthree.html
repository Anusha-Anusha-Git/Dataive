AI-powered Chest X-ray Diagnosis System

This deep learning project detects pneumonia from chest X-ray images using Convolutional Neural Networks (CNNs). Designed to assist radiologists and healthcare professionals, it offers a quick and efficient second opinion in diagnosing pneumonia cases.

Key Features

  • Real-time prediction from X-ray images
  • High accuracy CNN-based classification
  • Easy-to-use upload and result interface
  • Visualizations of model confidence & output

Technologies Used

  • Python, TensorFlow & Keras for model training
  • CNN architecture for image classification
  • OpenCV for image pre-processing
  • Matplotlib & Seaborn for visualization
  • Flask for backend API
  • Bootstrap 5 for frontend design
